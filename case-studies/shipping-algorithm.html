Shipping Algorithm

I was approached by a client with an existing web application that stored the shipping details for large transoceanic shipping containers. They were in need of validation on a form that ran the ISO 6346 algorithm on the BIC Code or serial number for the container. Such code needs to be registered at the Bureau International des Conteneurs in Paris to ensure uniqueness worldwide. Using Javascript I was able to add this validation to submit the form only after the ISO 6346 Algorithm was completed and the check number passed.

This was an exciting project to work on and opened my eyes to the complexities of international logistics and the role of technology in the industry.
