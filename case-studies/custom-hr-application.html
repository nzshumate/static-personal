Custom HR Application

I was approached by a medium sized manufacturing company to build a complex intranet that would act as an HR Management platform as well as an employment portal. I had been working with the PHP Framework Laravel in some personal projects, and thought it would be a perfect fit for this project. I began by designing out the Database Schema and scaffolding out the front end using Laravelâ€™s blade templating engine and the Bootstrap framework. The application consisted of several third party API integrations including, but not limited to, Twillo, Quickbooks, MS Outlook, Box, and others. Each one presented a unique set of challenges, and as I was the solo developer it allowed me to also expand my system architecture skills.

For the application environment I used the LEMP stack on a VULTR SSD cloud instance. One of the most challenging aspects to this project was configuring fillable PDF documents that would be populated with data collected via webform and stored in Box via the Box API.
