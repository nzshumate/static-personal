Zillow API Application

Similar to the Web Crawler script I wrote about previously, this API development pulled in real estate data for homes in North Carolina. However, instead of having to crawl the site and manually extract the data Zillow’s developers made an easy to understand API to search listings and pull the home’s information into a custom real estate search app. The design was already built for this application and it was my sole responsibility to write the API calls and make sure the response was parsed correctly and displayed in the application. I also built a simple CRUD GUI so that an admin could edit the results from the API call.

This project helped me to further my experience with working with complex API calls and displaying the data to the user in unique ways.
